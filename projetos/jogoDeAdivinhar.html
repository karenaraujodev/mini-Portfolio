Jogo de adivinhar

Sobre o Projeto

Um jogo interativo de adivinhar números criado com HTML, CSS e JavaScript, onde o usuário tenta descobrir secretamente um número gerado aleatoriamente pelo programa. A cada tentativa, o jogo responde dinamicamente se o palpite está correto ou não
